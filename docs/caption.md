## Caption

`polaris-caption` implements the [Polaris Caption component](https://polaris.shopify.com/components/titles-and-text/caption).

### Examples

Caption inline usage:

```hbs
{{polaris-caption text="Caption text"}}
```

Caption block usage:

```hbs
{{#polaris-caption}}
  Caption text
{{/polaris-caption}}
```
