## Inline error

`polaris-inline-error` implements the [Polaris Inline error component](https://polaris.shopify.com/components/forms/inline-error).

### Examples

Basic usage:

```hbs
{{polaris-inline-error
  fieldID="unique-ID"
  message="This username is already taken."
}}
```
