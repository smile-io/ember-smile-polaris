## Heading

`polaris-heading` implements the [Polaris Heading component](https://polaris.shopify.com/components/titles-and-text/heading).

### Examples

Basic usage:

```hbs
{{#polaris-heading}}
  This is a basic heading
{{/polaris-heading}}
```

Emphasised heading:

```hbs
{{#polaris-heading
  tagName="em"
}}
  This is an emphasised heading
{{/polaris-heading}}
```
